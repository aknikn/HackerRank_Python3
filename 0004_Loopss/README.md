# 0004: [Loops](https://www.hackerrank.com/challenges/python-loops/problem "https://www.hackerrank.com/challenges/python-loops/problem")
- level: Easy
- language: Python3


## Description of the problem

The provided code stub reads an integer, n, from STDIN.<br>
For all non-negative integers i＜n, print i^2.

### Example
The list of non-negative integers that are less than 3 is [0, 1, 2].<br>
Print the square of each number on a separate line.

> 0<br>
> 1<br>
> 4

### Input Format
The first and only line contains the integer, n.

### Constraints
1≦n≦20

### Output Format
Print n lines, one corresponding to each i.

### Sample Input 0
> 5

### Sample Output 0
> 0<br>
> 1<br>
> 4<br>
> 9<br>
> 16
